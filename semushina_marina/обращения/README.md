# Методология классификации

## Процесс работы

### 1. Анализ исходных данных
- 357 обращений за сентябрь-октябрь 2025
- Исходная тема "Запрос портала" не информативна

### 2. Разработка системы категорий
Иерархическая система из 5 групп по 4 подкатегории:

**A. Технические проблемы портала**
- A1: Регистрация, доступ, авторизация
- A2: Ошибки приложения/портала
- A3: Отображение данных
- A4: Технические неисправности связи

**B. Голосования и ОСС**
- B1: Недоступность опроса/голосования
- B2: Подтверждение прав для голосования
- B3: Процедура ОСС
- B4: Итоги голосований
